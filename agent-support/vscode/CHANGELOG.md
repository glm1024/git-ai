## [0.1.23]

- 新增 Git AI Activity Bar 的“数据上报”页面：从 Kilo 的本地设置补齐 Git AI 空字段，并允许用户维护上报服务器、组织与人员信息。
- 组织下拉选项由所填上报服务器加载，支持缓存、超时和失效值提示。
- 企业上报身份仅用于 Git AI 指标，不写入 Git Notes。
